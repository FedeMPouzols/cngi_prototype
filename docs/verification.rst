Verification
====================

Verification tests ensure that code matches its documentation, expected output and performance.

This section also serves as a de facto requirements guide as this package is meant to be functionally
equivalent to CASA6.  The requirements cross reference matrix may be viewed at
`<https://drive.google.com/open?id=1ihhoOFa-aRpCUnHe1eh8oL6fSTwtn6BFyz52yB5ko7A>`_

Visibility and Image regression sections demonstrate functional equivalence.
